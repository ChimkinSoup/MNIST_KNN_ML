# KNN Digit OCR

A simple, interactive digit recognition program using K-Nearest Neighbors (KNN) algorithm trained on the MNIST dataset. 

## Features

- Built-in MNIST .idx dataset loader
- Early-stopping KNN classifier for fast digit recognition
- Vectorized distance calculations for fast performance
- Interactive drawing canvas
- Real-time 28Ã—28 preview of what the model sees
- Keyboard shortcuts

## Running It 

1. Clone this repository:
```bash
git clone <repository-url>
cd KNN_Digit_OCR
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python Main.py
```

Once running, draw a digit and press enter to see the model's prediction
